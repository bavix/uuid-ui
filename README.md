# uuid-ui
Quick uuid conversion. Test Assistant

### Usage

It is recommended to use github pages, but it may not suit everyone. Run locally
```sh
git clone --depth 1 https://github.com/bavix/uuid-ui.git
cd uuid-ui
npm run build && npm run serve
```
